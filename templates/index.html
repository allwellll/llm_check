LLM LeetCode Benchmark

输入模型配置和 LeetCode Cookie，服务会在本机通过 leetcode-cli 依次执行题目、提交结果，并统计整体通过率。

如果第一次运行机器上没有 Rust 和 leetcode-cli，服务会自动安装，首次任务会更慢。

LeetCode Cookie 获取方法

  1. 先在浏览器登录 https://leetcode.com/。
  2. 打开开发者工具，进入 Application 或 Storage 页面。
  3. 在 Cookies 里找到 https://leetcode.com。
  4. 复制 csrftoken 和 LEETCODE_SESSION 的值。
  5. 这两个输入框也支持直接粘贴完整的 csrftoken=... 或 LEETCODE_SESSION=...; Domain=... 文本，服务会自动提取。
